# Support

Need help with `phenotype-journeys`?

- **Bugs and feature requests**: open an issue at
  https://github.com/KooshaPari/phenotype-journeys/issues — please use the
  bug report or feature request templates.
- **Security issues**: see [SECURITY.md](SECURITY.md) for the private
  disclosure policy. Do not file security bugs in public issues.
- **Questions / discussion**: use GitHub Discussions
  (https://github.com/KooshaPari/phenotype-journeys/discussions) for usage
  questions, integration help, and design discussion.
- **Maintainers**: the Phenotype ecosystem team. Response window is
  best-effort; this project is open source with no SLA.

For journey harness schema or verifier behavior, see
[`docs/journey-traceability.md`](docs/journey-traceability.md).
